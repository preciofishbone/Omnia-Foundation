# Omnia Foundation

Omnia Foundation is a product from Precio Fishbone. In order to use Omnia Foundation you need a product license for one of the following products:

* Omnia Foundation (stand-alone)
* Omnia Intranet
* Omnia Document Management
* Omnia QMS
